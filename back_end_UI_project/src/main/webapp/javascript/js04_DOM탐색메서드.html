DOM탐색메서드의 종류

1.getElementById("#idname"):태그의 id속성 이름으로 탐색 -반환값은 엘리먼트 1개

2.getElementsByName("name속성이름"):input 태그의 name속성 이름으로 탐색 -반환값은 엘리먼트 여러개(배열타입)

3.getElementsByTagName("tag 이름"):태그의 이름으로 탐색 -반환값은 엘리먼트 여러개(배열타입)
태그이름탐색

4.getElementsByClassName("class속성 이름"): class속성 이름으로 탐색 -반환값은 엘리먼트 여러개(배열타입)
클래스이름탐색

5.querySelectorAll("selector표현식"): 반환값은 엘리먼트 여러개(배열)
querySelector("selector표현식"):반환값은 엘리먼트 한개
셀렉터쿼리탐색
