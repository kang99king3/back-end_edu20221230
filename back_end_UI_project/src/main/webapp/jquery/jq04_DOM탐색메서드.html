필터 탐색

		eq(): 선택한 엘리먼트들 중에 인덱스로 탐색한다.
		slice():선택한 엘리먼트들 중에 인덱스 범위로 탐색할 수 있다.
		first():선택한 엘리먼트들 중에 첫번째 요소를 탐색한다.
		last():선택한 엘리먼트들 중에 마지막 요소를 탐색한다.

트리 탐색

		find(): 선택한 엘리먼트의 자손들 중에 탐색한다.
		children():선택한 엘리먼트의 자식요소를 탐색한다.
		parent()/parents("div"):선택한 엘리먼트의 부모요소를 탐색한다.
		prev(),prevAll()/next(),nextAll():선택한 엘리먼트의 앞/뒤에 오는 형제요소를 찾는다.
		siblings():선택한 엘리먼트의 형제요소를 모두 선택한다.
